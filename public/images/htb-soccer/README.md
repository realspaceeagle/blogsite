# HackTheBox Soccer - Image Assets

This directory contains image assets for the Soccer HTB writeup blog post.

## Required Images

Please add the following images to this directory:

1. **soccer-banner.png** - Main banner image for the blog post
2. **tiny-file-manager.png** - Screenshot of Tiny File Manager login
3. **root-shell.png** - Screenshot showing root shell access

## Image Guidelines

- Use PNG format for screenshots
- Optimize images for web (compress to reasonable file sizes)
- Include descriptive alt text in the markdown
- Ensure screenshots don't contain sensitive information

## File Naming Convention

- Use lowercase letters and hyphens
- Be descriptive: `nginx-config-discovery.png`
- Include sequence numbers if needed: `step-01-nmap-scan.png`
